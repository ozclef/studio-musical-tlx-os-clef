# 🎛️ Estudio de Producción Musical Ligero (Portable + Profesional)

## 🔹 Objetivo

Crear un sistema completo de producción musical:

* Ligero
* Portable (USB / PC de bajos recursos)
* Profesional (capaz de producir y vender servicios)

---

## 🔹 Filosofía

> No depender de software pesado ni loops prehechos.

Este setup está diseñado para:

* Crear música desde cero (MIDI)
* Tener control total del sonido
* Optimizar recursos (PC gama baja)

---

## 🔹 Estructura del Sistema

### 1. DAW Principal

**REAPER**

* Ultra ligero
* Portable
* Compatible con VST/VSTi

---

### 2. Instrumentos Virtuales (VSTi)

Plugins esenciales:

* Vital → Sintetizador moderno
* Dexed → Sintetizador tipo DX7
* LABS → Instrumentos reales (piano, strings, etc.)

Objetivo:

* Diseño de sonido
* Composición desde cero

---

### 3. Efectos (VST FX)

Tipos necesarios:

* EQ
* Reverb
* Compresor
* Delay

Objetivo:

* Mezcla profesional
* Control dinámico

---

### 4. Hardware mínimo

* Laptop (bajos recursos funciona)
* Teclado MIDI
* Audífonos / Monitores básicos

---

## 🔹 Flujo de Trabajo

1. Conectar teclado MIDI
2. Abrir REAPER
3. Cargar instrumento (VSTi)
4. Grabar MIDI
5. Editar notas
6. Aplicar efectos
7. Mezclar
8. Exportar
